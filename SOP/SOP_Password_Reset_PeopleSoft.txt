Title: PeopleSoft Password Reset SOP

Purpose:
To provide instructions for resetting a user's password in the PeopleSoft system.

Procedure:
1. Identify the user who requires a password reset from the incident details.
2. Extract the user's ID (user_id) from the request.
3. Use the `reset_peoplesoft_pwd` tool to perform the password reset.
   - Required Argument: `user_id` (The ID of the user requesting the password reset)
4. Note that this action requires human approval. The system will pause and ask for confirmation before executing the password reset.
5. Once approved, verify the tool execution result and update the incident accordingly.
